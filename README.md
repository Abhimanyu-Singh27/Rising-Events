# Rising Events Website

A modern full-stack event management website developed for Rising Events, designed to showcase event decoration services, event galleries, customer inquiries, and feedback management.

## Features

* Responsive and modern user interface
* Event services showcase
* Event gallery section
* Customer inquiry/contact form
* Feedback collection system
* WhatsApp integration
* Mobile-friendly design

## Tech Stack

### Frontend

* React
* JavaScript
* CSS

### Backend

* Node.js
* Express.js

## Project Structure

* Frontend application for user interaction
* Backend server for handling inquiries and feedback
* Gallery and service management

## Purpose

This project serves as the official online presence of Rising Events, helping customers explore services, view event portfolios, and connect with the team for event planning and decoration requirements.
